# 웹(프론트) ↔ Flutter 앱 기능 동등성 체크리스트

**기준**: 레퍼런스 프론트 `yuhyun-sensor-monitoring-front`, 백엔드 `yuhyun-sensor-monitoring-back`, 배포 API  
**검증 방식**: 각 항목은 릴리즈 전 실제 계정으로 한 번씩 시나리오 테스트할 것.

### 상태 표기

| 표기 | 의미 |
|------|------|
| ✅ | 구현·연동 완료로 코드 상 확인됨 |
| ⚠️ | 부분 구현 또는 동작은 있으나 웹 대비 범위·UX 차이 있음 |
| ❌ | 미구현 또는 미연동 |
| ➖ | 정책상 앱 미지원(웹 전용) — 백엔드/요구사항 사유 명시 |
